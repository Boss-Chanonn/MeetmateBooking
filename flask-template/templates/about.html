About MeetMate

Our Mission

MeetMate was developed as a student project to simplify the meeting room booking process. Our goal is to provide an easy-to-use platform for scheduling and managing meeting room reservations.

Features

  • Simple user registration and login system
  • Easy room booking interface
  • Dashboard to view and manage your bookings
  • Admin panel for managing users and rooms
  • Responsive design that works on all devices

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python Flask
  • Database: SQLite
  • Authentication: Session-based authentication
